1. Class
-class adalah template atau blueprint untuk membuat instance dari object 
-class mendefinisikan object 
-menyimpan data atau perilaku yang akan disebut property dan method 

contoh pembuatan class code : 

class Coba {

	//isi suatu class

}

Coba bebas namanya tapi diharuskan awal huruf besar



2. Object 
- object adalah implementasi atau instance dari rancangan class 
- beberapa object dapat dibuat dari satu class 

contoh code object :

class Coba {
	//isi
}

//membuat object instance dari class 
$a = new Coba() ; 
$b = new Coba() ;
